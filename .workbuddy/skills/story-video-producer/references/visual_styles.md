# Idiom Story Visual Style Guide

Each idiom has a unique mood — the visual style should match it.
Below are style categories, when to use them, and detailed prompt templates.

---

## Style Matrix

| Style | 关键词 | 适合成语类型 | 示例成语 |
|-------|--------|-------------|---------|
| **中国风水墨画** | ink wash, 水墨, 意境 | 神话/历史/大气 | 画龙点睛、愚公移山、精卫填海 |
| **卡通绘本风格** | storybook, 童趣, warm | 动物寓言/幽默 | 守株待兔、对牛弹琴、井底之蛙 |
| **皮影戏风格** | shadow puppet, 传统 | 民间故事/动作类 | 武松打虎、叶公好龙 |
| **剪纸风格** | paper cut, 民俗 | 节庆/民间传说 | 年年有余、喜上眉梢 |
| **工笔国画** | fine brushwork, 细腻 | 唯美/文学典故 | 沉鱼落雁、闭月羞花 |
| **Q版动漫** | chibi anime, 可爱 | 轻松/幼儿向 | 龟兔赛跑、狐假虎威 |
| **水彩绘本** | watercolor, dreamy | 温情/哲理 | 孟母三迁、铁杵磨针 |

## Detailed Style Templates

### 1. 中国风水墨画 (Chinese Ink Wash)

**适合**: 神话传说、历史典故、大气磅礴的成语

**基础 Prompt 模板**:
```
中国风水墨画风格，[主体描述]，[动作/场景]，
水墨晕染效果，留白构图，浓淡相宜，
古朴典雅，意境深远，4K画质
```

**色彩基调**: 黑白为主，赭石/花青点缀
**运镜建议**: 缓慢推进、少切换，配合水墨的"静"感

**示例 — 愚公移山**:
```
Scene 1: 中国风水墨画风格，太行王屋两座大山巍峨耸立，
         云雾缭绕山腰，山下一个小村落隐约可见，气势恢宏

Scene 2: 中国风水墨画风格，白发老翁愚公带领儿孙挥锄挖山，
         人物虽小但动作坚定，山石飞溅，水墨笔触苍劲有力

Scene 3: 中国风水墨画风格，邻居智叟路过嘲笑愚公不自量力，
         双手叉腰神态轻蔑，愚公停下锄头转身回答，对比强烈

Scene 4: 中国风水墨画风格，天帝被愚公精神感动，
         派两位大力神下凡搬走大山，神人衣袂飘飘，山体缓缓移动

Scene 5: 中国风水墨画风格，山路变通途，村民欢庆，
         大片留白中一条小路蜿蜒向前，阳光穿透云雾洒落大地
```

### 2. 卡通绘本风格 (Cartoon Storybook)

**适合**: 动物寓言、幽默讽刺、面向低龄儿童

**基础 Prompt 模板**:
```
温馨可爱的卡通绘本风格，[主体描述]，[动作/场景]，
柔和的色彩，圆润的角色造型，故事书插画质感，
温暖的光线，毛茸茸的质感，children's book illustration
```

**色彩基调**: 暖色调为主，粉彩/马卡龙色系
**运镜建议**: 活泼切换，配合快节奏旁白

**示例 — 守株待兔**:
```
Scene 1: 温馨可爱的卡通绘本风格，胖乎乎的农夫在田地里挥汗劳作，
         阳光明媚，绿色田野一望无际，小动物在远处探头探脑

Scene 2: 温馨可爱的卡通绘本风格，一只毛茸茸的白兔子慌慌张张跑过来，
         一头撞在大树桩上，眼冒金星，晕乎乎地倒在旁边

Scene 3: 温馨可爱的卡通绘本风格，农夫惊喜地抱起兔子，
         脸上笑开了花，旁边树桩特写，树桩上还有兔子撞出的凹痕

Scene 4: 温馨可爱的卡通绘本风格，农夫每天坐在树桩旁等待，
         从春天到冬天，四季变换，树桩长出了青苔，田地却荒芜了

Scene 5: 温馨可爱的卡通绘本风格，农夫饿得肚子咕咕叫，
         看着荒废的田地后悔不已，远处其他农夫正在丰收，
         鲜明的对比
```

### 3. 皮影戏风格 (Shadow Puppet)

**适合**: 动作感强的民间故事、英雄传奇

**基础 Prompt 模板**:
```
中国传统皮影戏风格，[主体描述]，[动作/场景]，
半透明皮质质感，关节可动，暖黄背光，
镂空雕刻纹样，投影在米色幕布上，古朴民俗感，flat design
```

**色彩基调**: 暖黄+深褐，半透明质感
**运镜建议**: 横向移动为主，模拟皮影戏舞台

**示例 — 叶公好龙**:
```
Scene 1: 中国传统皮影戏风格，叶公在厅堂中欣赏满墙的龙纹装饰，
         雕梁画栋处处是龙的图案，暖黄灯光映照，精致典雅

Scene 2: 中国传统皮影戏风格，叶公向客人炫耀自己的龙纹收藏，
         神态得意，手指墙上龙画，客人点头称赞

Scene 3: 中国传统皮影戏风格，天上真龙听到叶公如此喜爱自己，
         从云端探下巨大的龙头，鳞片发着金光，向叶公家飞去

Scene 4: 中国传统皮影戏风格，真龙从窗户探进头来，
         叶公吓得魂飞魄散跌坐在地，茶杯打翻，表情极度惊恐

Scene 5: 中国传统皮影戏风格，真龙失望地飞回天上，
         叶公瘫坐地上瑟瑟发抖，字幕：原来叶公爱的是假龙，不是真龙
```

### 4. 剪纸风格 (Paper Cut)

**适合**: 节庆成语、民俗主题、对称构图

**基础 Prompt 模板**:
```
中国传统剪纸艺术风格，[主体描述]，[动作/场景]，
红色纸张质感，镂空雕刻效果，对称构图，
民间艺术感，喜庆吉祥，层次分明，papercut art
```

**色彩基调**: 中国红为主，可配金/黄点缀
**运镜建议**: 静态对称画面+局部放大切换

### 5. Q版动漫 (Chibi Anime)

**适合**: 幼儿向、轻松搞笑的成语

**基础 Prompt 模板**:
```
可爱Q版动漫风格，[主体描述]，[动作/场景]，
大头小身体萌系造型，明亮鲜艳的色彩，
表情夸张可爱，干净的线条，白色背景简洁，2D animation look
```

**色彩基调**: 明亮饱和，高对比度
**运镜建议**: 快速切换、夸张转场

**示例 — 狐假虎威**:
```
Scene 1: 可爱Q版动漫风格，一只圆滚滚的橘色大老虎在森林里散步，
         小动物们吓得四散逃跑，老虎一脸傲娇

Scene 2: 可爱Q版动漫风格，一只灵巧的小狐狸从树后跳出来，
         对着老虎笑眯眯说话，老虎歪着头半信半疑

Scene 3: 可爱Q版动漫风格，小狐狸大摇大摆走在前面，
         大老虎乖乖跟在后面，小动物们看到老虎都吓跑了

Scene 4: 可爱Q版动漫风格，小狐狸得意地回头对老虎眨眼睛，
         老虎这才恍然大悟被骗了，气得头顶冒烟

Scene 5: 可爱Q版动漫风格，小狐狸逃到树上咧嘴笑，
         老虎在树下气鼓鼓地转圈，画面定格在搞笑的一幕
```

## Style Selection Decision Tree

```
成语主角是？
├── 神话人物/神仙 → 中国风水墨画
├── 动物 → 
│   ├── 幽默搞笑 → Q版动漫 或 卡通绘本
│   ├── 寓意深刻 → 水彩绘本
│   └── 民间传说 → 皮影戏风格
├── 历史人物 →
│   ├── 大气磅礴 → 中国风水墨画
│   ├── 唯美文学 → 工笔国画
│   └── 幼儿启蒙 → 卡通绘本
├── 自然景物 → 水彩绘本 或 水墨画
└── 民俗节庆 → 剪纸风格

成语情绪是？
├── 激昂/励志 → 水墨画、工笔国画
├── 温馨/感动 → 水彩绘本
├── 幽默/讽刺 → 卡通绘本、Q版动漫
├── 恐怖/警示 → 皮影戏（暗调）
└── 喜庆/祝福 → 剪纸风格
```

## Prompt Engineering Tips

1. **场景描述越具体越好**: 不只要说"一个人"，要精确到"白发苍苍的老翁拄着拐杖"
2. **同一故事内风格一致**: 所有 5 个场景用同一套风格关键词
3. **情绪关键词**: 每个场景加上氛围词（宁静的/紧张的/欢快的/震撼的）
4. **避免现代元素**: 成语故事场景中不要出现现代建筑、服饰、物品
5. **动作描述用进行时**: "正在挥笔"比"画龙"更有画面感

## 输出规格建议

| 风格 | 画面比例 | 色调 | 说明 |
|------|---------|------|------|
| 水墨画 | 16:9 | 黑白+单色点缀 | 1080p 足够，4K 更好 |
| 卡通绘本 | 16:9 | 暖色调柔和 | 饱和度不宜过高 |
| 皮影戏 | 16:9 | 暖黄+深褐 | 注意半透明效果 |
| Q版动漫 | 16:9 | 明亮鲜艳 | 线稿清晰为佳 |
